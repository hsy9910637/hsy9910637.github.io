健身

发表于 2019-07-18 更新于 2019-07-23 分类于 生活

昨天晚上终于下定决心要开始我的减肥大业，报了个私教课。我的数据真的是惨不忍睹

这是之前我收藏的一些饮食建议

以下是教练对于我健身的一些建议

饮食

  1. 低碳水（米饭、面条、包子少吃这些都是很容胖的高碳水，最好早上和中午吃）吃精瘦肉、海鲜，并不会胖。主要是碳水化合物吃的太多） 按照我的体重一天的主食先按照体重KG/2-3克 =138克碳水 ，吃100克米饭=25克碳水化合物 ，一天总共的主食是550克 分成三餐就是每餐183克米饭或者杂粮，每个人的碳水身体反应吸收不一样，要是体重不明显下主食的量在一点点降 。高蛋白每天体重KG/0.8-1.0克蛋白质需要至少克蛋白质 100克鸡胸=25克蛋白质 一天的肉或者海鲜总量要吃270克（多吃海鲜、牛肉、鸡胸肉低脂肪的）绿叶的蔬菜可以大量的吃 （蔬菜属于碱性食物可调节身体酸碱平衡和代谢肌肉酸痛）下午饿了加餐苹果，或猕猴桃，各种低果糖的水果、全麦面包。三餐一天的总量就按照1.低碳水（550克主食）
  2. 高蛋白食物（270克海鲜或精肉），
  3. （各种蔬菜多吃） 配合少许鱼油，维生素。

会胖的主要三个原因：

  • 第一个碳水化合物吃太多了f里面都是淀粉，过多的淀粉会转化成脂肪储存。
  • 第二个是糖，酸奶的糖、水果果糖太高超过每日人体所需都会转化成脂肪储存（西瓜、葡萄少吃太甜的水果），
  • 第三个，一天的总体食入热量太高高出每天的总消耗会胖，就算是吃健康的饮食也会，热量太高！
